# map

## 项目介绍
一个基于echarts的可视化地图,在地图上添加柱状图统计数据,最小可以统计到县级区级
vue@2
echarts@4.9.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
